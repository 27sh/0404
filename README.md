# 16비트의 데이터를 2진수로 출력하는 프로그램

<stdint.h>를 이용합니다.

scanf를 이용하여 16진수 입력을 받아 uint16_t 에 저장합니다.

mask를 사용하여 입력 데이터를 추출하고 2진수로 출력합니다.

4 또는 7에 콜론을 출력하고

비트가 0이면 0을 출력하고, 그렇지 않으면 1을 출력합니다.
